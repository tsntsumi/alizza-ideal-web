## あなたのホームページは、仕事をしてますか？

<div style={{float: "right", width: "30%",
	     minWidth: "128px", marginLeft: '1em',
	     padding: "0 var(--borderSpacing) 0 0"}}>
<GatsbyImage alt="Seles Person"
	     image={props.images[0]}/>
</div>

せっかく作ったホームページに、こんな疑問やお悩みをお持ちのかたが大勢います。

- せっかくホームページを作ったのに、誰からも問い合わせがない
- 業者にいわれるがまま SEO 対策したけど、検索上位に上がってこない
- 毎月払っている保守費って、なにに使われているのだろう？

でももし、あなた専属の営業担当のように、
ホームページを働かせられるとしたらどうでしょうか。

<clear/>

## ネットで働く営業担当のホームページとは？

優秀な営業担当のようなホームページなら、
ホームページのよくある悩みが解決でき、
こんな効果を期待できます。

* ホームページからの問い合わせが安定
* しかも、成約率がやばい
* 毎月の保守にかかる作業が明確
* 集客はホームページにまかせられるので、お客さんにかける時間が増加
* その上、プライベートの時間も増加
* だから、お客さんも満足、あなたもあなたの家族も充実

そんなホームページは、あります。

そんな **公式サイト** 型ホームページは、
専門的な知識と技術を提供し、
信頼と真心をもってお客さんに接する、
**士業** の方には特にオススメです。

## 公式サイトってどんなホームページ？

<div style={{float: "right", width: "30%",
	     minWidth: "128px", marginLeft: '1em',
	     padding: "0 var(--borderSpacing) 0 0"}}>
<GatsbyImage alt="Attract Customers"
	     image={props.images[1]} />
</div>

今回ご紹介する公式サイトは、ホームページ制作会社が作る、
単なるかっこいいホームページとは異なります。

ホームページ制作会社は、ホームページをデザインするプロですが、
集客や売上を上げるプロではありません。

ですから、ここで説明する公式サイトを作る秘訣を読めば、
あなたが主体性を持ってホームページの制作を依頼できるようになります。

<clear/>

## ホームページに期待することはなんですか？

多くの方が、COVID-19が沈静化している今がチャンスと、
新規のお客さんを集めるために宣伝に精を出しています。

ただ、無料でできるSNSやブログを始めたものの、
時間ばかり取られて、
利益につながる本来のサービスがおろそかになりそう、
と多くの人が危機感を覚えています。

でも、公式サイト型ではない、
単なるホームページを作った人は、期待通りの結果を得られていません。

### もし、公式サイトを作れば・・・

集客活動の手間と時間が減り、
自分の業務に集中することができるようになります。

それによって、売り上げも利益もプライベートの時間も、
今よりもっと多く手に入れることができるでしょう。

## なんで公式サイトはそんなことができるの？

なぜなら、お客さんが購入したり契約したりするときの心理状態に基づいて、
公式サイトを設計するからです。

<div style={{float: "right", width: "60%",
	     minWidth: "256px",
	     margin: '1em 0 0 1em', padding: "0 var(--borderSpacing) 0 0"}}>
<GatsbyImage alt="Attract Customers"
	     image={props.images[4]} />
</div>

大雑把に言うと、ホームページを見た人を、

1. 惹きつけ、
2. 興味関心をもたせ、
3. 信用、信頼させ、
4. 安心させ、
5. 問い合わせや依頼をしていただく。

というようになる順番で構成するのです。

<clear/>

多くのホームページでは、このような順番になっていないか、
一部しか書いていません。

というのも、多くのホームページ制作会社では、
「A4サイズあたりに換算していくら」と料金設定しているせいです。
あなたのご予算ではこのくらいしか作れません、というわけです。

しかし、ホームページの値段はスマホやパソコンのアプリと同様に、
目的達成のための機能に基づいて対価を決めるべきです。

目的を達成できなければ、
いくらカッコよくても料金を請求できるだけの価値はありません。

## なぜその順番が重要ですか？

よくあるホームページでは、いきなりサービスの説明に入ったり、
カッコいいイメージを表示します。

しかし、お客さんとの対話を、合コンに例えてみましょう。

最初の自己紹介のときに、いきなり「付き合ってください」とか
「この後ヒマ？」なんていいだしても、よい結果は得られません。

自分のスペックを、こんな風にずらずらと並べたりしても、
興味を持ってもらえません。

- 年収がいくらとか、
- こんなに貯金があるとか、
- どんな大企業に勤めているとか、
- どこそこの有名大学を出たとか、
- いくらの車に乗っているとか、
- どれだけ高価なブランドの時計や服やバッグをもっているとか、
- 俺にはこんな友達がいるとか、
- あいつは俺が育てたとか・・・。

いわゆる自慢話、売り込みですね。

あなたも、なんとなく開いたホームページに、
いきなりサービスの内容や機能や特徴が
どれほどよいか書いてあるのをみたらどうでしょうか。

よっぽど興味がないかぎり、
読み続けたりしないのではないでしょうか。

ホームページでも、説明する順番が大切なのです。
お客さんが、今どんな気持ちでいるかを考えて、
売り込まないようにすることが重要なんです。

<clear/>

## でも、こんなことはできません

<div style={{float: "right", width: "30%",

	     padding: "0 var(--borderSpacing) 0 0"}}>
<GatsbyImage alt="Hypnosis"

しかし、アドラー心理学でもいうように、
人の信念や行動を変えることはできません。
ほしくもない人に、売りつけることはできないのです。

そんなことをしたがるのは、詐欺師とか、なんとかビジネスの勧誘くらいです。

人を操ったり、洗脳することはできません。

ですので、人を騙してでも物やサービスを売りたいという方には、
公式サイトは向いていません。

<clear/>

## ではなぜ公式サイトは成果を出せるの？

<div style={{float: "right", width: "30%",
     	     minWidth: "128px", marginLeft: "1em",
	     padding: "0 var(--borderSpacing) 0 0"}}>
<GatsbyImage alt="Funnel Filter"
	     image={props.images[2]} />
</div>

では、なぜ公式サイトは、お客さんをコントロールも
洗脳もしないのに成果を出しやすいのでしょうか。

それは、公式サイトに訪れた多くの人の頭の中から、
サービス商品に対する漠然とした不安、疑問、
疑念、躊躇を、読み進めるうちに段階的に取り除いていきます。

そして、最終的にあなたのサービス商品への欲求が高まるようにするのです。

読み進めていって、自分には必要ないと感じた人は、
そこで読むのをやめて去っていくことでしょう。

つまり、公式サイトはろ過器（ファネル; Funnel）のように、
お客さんの購入意欲を純化していくのです。

もしかするとあなたは、

- それじゃあお客さんが少ししか集まらない！
- いろんな人がいるのだから、いろんな人にアピールしたい！

と思うかもしれません。

でも、しかし・・・。

<clear/>

## 単なるホームページを作った会社の末路

たしかに、世間一般によくあるホームページは、
そのような売り手側の要望におこたえして、
より多くの人にアピールするようにデザインされています。

しかし、そんなホームページですが、、、

中小企業庁の調べによると、
ホームページからの新規受注がよくあるという企業は、
７２１社中たったの 1.9% (約１４社）だけなんです。
（中小企業白書より）

また、作ったホームページが有効だったと答えたのは、
５０５社のうちの 5.1%（約２６社）しかありませんでした。

つまり、ホームページで集客できたのは、
１００社のうちたったの２社。

逆に、１００社のうち９５社の社長さんは
よくあるホームページを作って後悔しているのです。

## なぜ大勢ではなく、少人数にアピールするほうがいいの？

それは、あなたもご存知のように、
大きなつづらではなく、小さなつづらの方に宝物が詰まっているからです。

そして、数は少なくても貴方のサービスに
ピッタリあったお客さんなら、
無理をいってクレームをつけたり、値切ったり、
結果に不平不満を言ったりしない、理想的なお客さんなのです。

つまり、集客活動に時間をかけなくてもよくなるだけでなく、
クレーム処理にも時間をかけなくてもよくなるのです。

### 先程の合コンの例えを思い出してください。

あなたは、全員にいい顔している人より、
自分だけを見てくれている人のほうに、
好意をもちませんか？
